Opdracht 1

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Antwoord

  • Met pseudo elementen kunnen gedeelten van elementen gestyled worden. Een voorbeeld hiervan is het stylen van de eerste regel van een tekst. De Syntax van een Pseudo element is als volgt: selector::pseudo-element "{" property: value; "}" voorbeelden van pseudo elementen zijn:
    • :First-line, waarbij de eerste zin van een tekst wordt gestyled.
    • :First-letter, waarbij de eerste letter van een element wordt gestyled.Dit geldt overigens alleen voor block elementen.
    • :Before, waarbij content voor een element wordt geplaatst.
    • :After, waarbij content achter een element wordt geplaatst.
    • :Selection, waarbij de geselecteerde text wordt gestyled.
