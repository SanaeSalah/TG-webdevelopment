Opdracht 1

Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

Antwoord

Positition property specificeert een type van de positie mehode dat gebruikt voor een element.

  1. Een element met static property is niet gepositioneerd op een speciale manier, maar volgens de normale fllow van de pagina (Default value, dus top, right, bottim en left properties hebben geen invloed).
  2. Een element met relative property is relatief gepositioneerd t.o.v. de normale positie. Het gebruik van de top, right, bottim en left properties heeft als gevolg dat het aangepast wordt.
  3. Een element met fixed property is relatief gepositioneerd t.o.v. de viewport. Dit houdt in dat zelf als je de pagina naar beneden scrolt dat het niet mee beweegt. Het blijft op dezelfde plek staan en laat geen ruimte achter. Het gebruik van de top, right, bottim en left properties heeft als gevolg dat het aangepast wordt.
  4. Een element met absolute property is relatief gepositioneerd t.o.v. de vorige voorouder element. Als het geen voorouder element heeft, dan wordt de body van het document gebruikt. Hierdoor staat niet vast en gaat het element gaat mee indien gescrold wordt. Het kan alle posities aanemen behalve static.
  5. Een element met sticky property is relatief gepositioneerd t.o.v. scrol positie van de gebruiker. Het blijft sticken op een plek. En gaat mee met tijdens het scrollen.Het gebruik van de top, right, bottim en left properties heeft als gevolg dat het aangepast wordt.
