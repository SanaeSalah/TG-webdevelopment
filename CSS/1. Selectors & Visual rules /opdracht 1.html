Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Antwoord

Er zijn drie manieren om CSS te inporteren in HTML. Deze drie manieren zijn: Met een externe stylesheet kan het uiterlijk van een hele website veranderen door slechts één bestand te wijzigen! Elke HTML-pagina moet een verwijzing bevatten naar het externe stijlbladbestand in het "link" -element, in de head-sectie.Externe stijlen worden gedefinieerd in het "link"-element, in het "head" -gedeelte van een HTML-pagina. Het moet opgeslagen als .css en er mogen geen HTML-tags in staan. Een interne stylesheet kan gebruikt worden een enkele HTML-pagina een unieke stijl heeft. De interne stijl wordt gedefinieerd in het "style"-element, in de head-sectie. Interne stijlen worden gedefinieerd in het "style"-element, in het "head"-gedeelte van een HTML-pagina. Inline style kan gebruikt worden door een style toe te passen op een element door een style-attribuut toe te voegen.
