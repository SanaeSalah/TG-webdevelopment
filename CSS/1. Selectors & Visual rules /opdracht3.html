Opdracht 3

Voorbeelden van verschillenden selectors

Antwoord

Een type selector komt overeen met de naam van een type documenttaalelement. Een type selector komt overeen met elk exemplaar van het elementtype in de documentboom. Een voorbeeld hiervan zijn alle h1-elementen of a-elementen in een document. Het selecteert alle elementen van het gegeven type binnen een document. De descendant selector komt overeen met alle elementen die afstammen van een opgegeven element. Een voorbeeld hiervan zijn de "p"-elementen binnen "div"-elementen. Een voordeel is dat een specifieke element gestyld kan worden. De .class-selector selecteert elementen met een specifiek class-attribuut. Om elementen met een specifieke klasse te selecteren, schrijft u een punt (.) -Teken, gevolgd door de naam van de klasse. Een voorbeeld hiervan is .Text. Alle elementen binnen een class gestyled kunnen worden.
