Opdracht 3

Wat gebeurt er als je een width en een height toevoegt aan de span tag ( <span> ) van vorige opdracht?

Antwoord

De span tag wordt gezien als een inline block. Een kenmerk van een inline blocks is dat de breedte of hoogte niet aangepast kunnen worden. Met andere woorden zal het aanpassen van de hoogte en breedte niets opleveren.
