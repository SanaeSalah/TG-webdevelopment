Opdracht 7

Maak nu een blok met width: 100px, height: 100px, margin: 5px, padding 5px en border-width: 5px. Zie hieronder:

Dit is de content
  1. Hoe breed is de box?
  2. Hoe hoog is de box?
  3. Wat merk je op met width en height?

Antwoord 7

  1. Totale widht is 2 * border width + 2 * padding + width = 2*5px + 2*5px + 100px = 120px
  2. Totale height is 2 * border width + 2 * padding + height =2*5px + 2*5px + 100px= 120px
  3. Zowel de hoogte als de breedte hebben de zelfde waarde, wat ook logisch is aangezien de box de vorm van een vierkant heeft. Bovendien is de totale hoogte en breedte niet gelijk aan 100 px, want de margin en border zijn niet meeberekend. Width = breedte van de content en height = hoogte van de content, exclusief borders of paddings.
