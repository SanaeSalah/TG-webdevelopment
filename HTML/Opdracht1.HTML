Onderdeel 1 - HTML Opdrachten

1. Wat is het verschil tussen het internet en het world wide web?

Het internet is een wereldwijd systeem dat computernetwerken over de hele wereld met elkaar verbindt. World Wide Web is een groot verzameling van webistes en maakt deel uit van het netwerk.

2. Wat is het verschil tussen HTML, CSS en JavaScript?

HTML is een opmaaktaal voor het maken van een website en vormt de basis van elke website. CSS daarentegen omvat de opmaak en indeling van je website (layout). Het is dus een stijlbladtaal. JavaScript is een scriptaal en kan gebruikt worden om HTML en CSS dynamische te maken. JavaScript is geavanceerd en biedt meer functies en interacties voor een webpagina. HTML, CSS en JavaScript vormen samen de front-end van een website.

4. HTML gebruikt basis tags om structuur op te bouwen. De html, head en body. Geef voor elke element aan waarvoor het dient.

Het HTML-element representeert het element op het hoogste niveau van een HTML-document. Het head-element beschrijft de data (document title, character set,styles, scripts) die niet zichtbaar is. Het geeft dus informatie over het document. Het body-element beschrijft de content van het HTML document.

5. Op HTML elementen kun je attributen gebruiken. Wat zijn HTML attributen ? Geef minimaal 5 attributen en maak voorbeelden waar je de attributen in toepast.

Een atribuut is een specificatie (aanvullende informatie) van een element. Voorbeelden hiervan zijn:

"img alt="text" geeft een afbeelding met een alternatieve tekst weer.
"class" definieert een class-naam en geeft de mogelijkheid om bepaalde eigenschappen aan een element te koppelen.
"Href" geeft de bestemming weer.
"a" beschrijft een link.
"id" geeft de id-naam weer.
"Src" beschrijft de plaats van de afbeelding, audio en of video.

6. Waarom is een goede HTML Structuur belangrijk? Is deze code een goede structuur? Leg uit.

Een goede structuur is van belang, zodat de de bezoeker van de webpagina weet waar hij de juiste informatie snel kan opzoeken. Geen goede structuur, geen elementen gebruikt. Website zal er onoverzichtelijk uitzien.

7. Wat is het verschil tussen inline en block elementen?

Een block-level element begint altijd op een nieuwe regel en neemt de volle breedte van het browservenster in. Het strekt zich naar links en rechts uit zo ver als het kan. Een inline element start niet op een nieuwe regel.

9. Sinds de komst van HTML5 zijn er elementen die betekenissen hebben. Wat wordt er bedoeld met HTML semantics? Waarom is het gebruik van semantics belangrijk?

Met een semantisch web kunnen gegevens worden gedeeld en hergebruikt tussen applicaties. Schematische element beschrijft duidelijk wat de betekenis is en is voor zowel de browser als developer helder. Het helpt om de de structuur van de code makkelijker te beschrijven.

10. Wat voor content zou je in de volgende elementen gebruiken? Geef in je antwoord voor elk element een toelichting.

● "head": Beschrijft de data (document title, character set,styles, scripts) die niet zichtbaar is. Het geeft dus informatie over het document.
● "footer": Beschrijft de voettekst (copyright informatie, contactgegevens, auteurschap informatie).
● "tbody": Beschrijft de dominante inhoud van de body (tabel).
● "nav": Beschrijft een navigatie link (navigatiebalk).
● "section":Beschrijft een sectie/deel van een document.
● "aside": Definieert een deel van de inhoud naast de inhoud waarin deze is geplaatst.
● "article": Specificieert onafhankelijk, voorbeeld is een blog.

11. Onderzoek en leg uit waarom headers zowel belangrijk zijn voor zoekmachines als voor de gebruikers.

Headers zijn afbeeldingen op je Homepagina, die ervoor zorgen dat mensen je website aantrekkelijk vinden. Uit onderzoek blijkt namelijk dat 70% van de website bezoekers afhaakt als ze de header van een website niet aantrekkelijk vinden, of als de header niet aansluit bij de rest van de website. Het is dus belangrijk om hier genoeg aandacht aan te besteden en te zorgen dat de header écht aansluit bij de content op je website.

18. Onderzoek welke 4 attributen een link tag kan hebben, leg uit wat deze doen en voeg deze als voorbeeld toe op je website.

● Met rel attribuut wordt de relatie tussen het huidige document en het gekoppelde document aangegeven.
